# Copy-Cat — Releases

**Copy-Cat** gives your clipboard 9 lives: nine keyboard-driven clipboard slots plus pinned items, without changing the copy/paste habits you already have.

This repository hosts the downloadable builds. The source code lives in a private repository.

## Download

Grab the latest build from the [Releases page](https://github.com/fuck-eer/copycat-releases/releases/latest):

| Platform | File | Status |
|----------|------|--------|
| macOS 12+ | `copy-cat-setup-mac.dmg` | Beta |
| Windows 10+ | `copy-cat-setup-windows.exe` | Coming soon — [join the waitlist](https://copycat.example.com) |

Or get it from the landing page, which tracks beta seats: **[Copy-Cat website](https://copycat.example.com)**
